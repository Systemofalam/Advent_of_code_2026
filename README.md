# Advent of Code 2026

Solves the Advent of Code 2026 calendar.  
It's in Python because I'm tired this year :p

## Structure

- One folder per day: `Day01`, `Day02`, ..., `Day25`
- Each day usually has:
  - `DayXX.py` – solution script
  - maybe some scratch / extra files if the puzzle was annoying 

## How to run

From a given day folder, e.g. Day05:

```bash
python3 Day05.py input.txt
